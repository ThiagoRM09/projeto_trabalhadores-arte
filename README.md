# 📜 Os Trabalhadores – Website

Projeto desenvolvido para a disciplina de **Artes do Ensino Médio**, no qual o grupo criou uma **história ilustrada sobre o trabalho ao longo do tempo**.

Como também fazemos parte do curso técnico em **Desenvolvimento de Sistemas**, decidimos expandir o projeto e desenvolver **um site interativo** para apresentar a narrativa criada pelo grupo.

---

# 🎯 Objetivo do Projeto

O objetivo do projeto é representar **a evolução do trabalho humano**, comparando diferentes períodos históricos e refletindo sobre como o trabalho mudou ao longo do tempo.

A ideia foi unir **arte, história e tecnologia**, transformando uma simples ilustração em uma experiência digital através de um website.

---

# 💻 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

- **HTML5** → estrutura das páginas
- **CSS3** → estilização e layout do site
- **JavaScript** → interações da página
- **Git e GitHub** → versionamento do projeto

# 🌐 Estrutura das Páginas

## 🏠 Home
Página inicial do site, responsável por apresentar o tema **"Os Trabalhadores"** e introduzir o visitante ao projeto.

## ⏳ Passado
Página que apresenta **como era o trabalho antigamente**, com elementos visuais e referências históricas.

## ⚙️ Presente
Página que mostra **como o trabalho acontece atualmente**, destacando mudanças sociais e tecnológicas.

---

# 🎨 Elementos Visuais

O site utiliza diferentes elementos visuais para reforçar o tema do projeto:

- imagens históricas
- gifs animados
- fundos temáticos
- gradientes
- elementos relacionados ao trabalho e ao tempo

Esses recursos ajudam a transmitir visualmente a **transformação do trabalho ao longo das épocas**.
